# Code Review Skill

帮助进行代码审查的技能。

## 使用场景

在提交代码前，让 Claude 审查代码质量。

## 提示词

```
请审查我刚才修改的代码，检查：

1. 代码质量
   - 是否有明显的 bug
   - 是否有性能问题
   - 是否有安全风险

2. 代码风格
   - 是否与项目风格一致
   - 命名是否清晰
   - 是否有适当的注释

3. 最佳实践
   - 是否遵循设计模式
   - 是否有重复代码
   - 错误处理是否完善

请给出具体的改进建议。
```

## 示例对话

**你：** 我修改了 user-service.ts，请帮我审查代码

**Claude：** [执行上述审查流程并给出反馈]
